Abyssfall
Dark Fantasy Action‑RPG / Roguelite
Unity Version: 6000.3.15f1 LTS

🧩 Overview
Abyssfall is a dark‑fantasy, isometric Action‑RPG / Roguelite focused on deep build‑crafting, procedural vertical exploration, and long‑term progression.
Players descend through endlessly mutating underground layers, adapting to Depth Pressure, corruption, mutators, and hostile biomes.
The game emphasizes modular weapon evolution, emergent gameplay, and high replayability.

🎮 Core Fantasy
Descend into the impossible depths.
Mutate, adapt, survive.
Forge a build capable of reaching places no one else has ever seen.

🏛️ Project Vision
Abyssfall aims to be a long‑lasting, system‑driven RPG where:

the world mutates as the player descends,

the player’s build evolves through corruption, mutators, and artifacts,

every run creates unique emergent stories,

the ultimate goal is to reach record‑breaking depths.

⚔️ Key Features
Modular Weapon System
One weapon, infinitely customizable

Unlimited enchant stacking

Mutations that change movesets and behavior

Synergy with skills, artifacts, and corruption

Corruption & Mutation
Depth Pressure increases corruption

Corruption grants powerful boons and dangerous flaws

Mutations affect combat, exploration, and world interaction

Procedural Vertical World
World generated downward in Tiers

Each Tier has unique biomes, enemies, mutators, and rules

Chunk‑based procedural generation

Archetypes: caverns, ruins, mechanical depths, flooded cities, fractal spaces, eldritch realms

Emergent Gameplay
Systems interact to create unpredictable scenarios

Mutators + corruption + biome rules = unique challenges

Player stories emerge from gameplay, not scripts

Base & Meta‑Progression
Build and upgrade base structures

Unlock rituals, crafting, workshops, NPCs

Permanent progression between runs

Adaptive Enemies & Bosses
Modular enemy design (parts, tags, behaviors)

Bosses mutate based on active mutators and player build

Eldritch forms and corruption‑based transformations

Endgame: The Endless Abyss
Infinite depth scaling

Seasonal leaderboards

Legendary runs with unique mutators and bosses

🗺️ World Structure (Tiers)
Tier	Depth	Description
1	0–20 m	Natural caves, ruins, low corruption
2	20–50 m	Catacombs, traps, early machines
3	50–120 m	Mechanical depths, energy fields
4	120–250 m	Subterranean oceans, flooded ruins
5	250–500 m	Fractal anomalies, warped geometry
6	500+ m	Eldritch realms, extreme corruption


🧪 Core Systems
Combat
Light/heavy attacks

Stamina‑based action economy

Dodges, parries, positional bonuses

Weapon‑dependent movesets

Skills & Attributes
Skill list instead of classic tree

Adept → Expert → Master progression

Attribute gates (Strength, Dexterity, Will, Endurance)

Tags define synergies (melee, bleed, echo, ritual, mobility)

Crafting
Blueprint‑based

Weapon modules, runes, enchantments

Alchemy, stabilizers, corrupted materials

Crafting tied to character progression

Procedural Generation
Chunk‑based

Archetypes + biomes + mutators

Event system (rituals, anomalies, ruins, minibosses)

Difficulty scales exponentially with depth


Factions: Depth Wardens, Ritual Cult, Rune Traders.
